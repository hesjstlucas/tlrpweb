# TLRP Landing Page

Standalone static site for a TLRP homepage.

## Files

- `index.html` contains the page structure and copy.
- `styles.css` contains the layout, colors, and responsive styling.
- `assets/tlrp-mark.svg` is the custom TLRP logo.

## Quick edits

- Replace the Discord invite in `index.html`.
- Update any section wording to match your final server details.

## Upload to GitHub

1. Create a new empty GitHub repository.
2. Push this folder to that repository.
3. If you want a live site, enable GitHub Pages from the repository settings.
